An AI-powered career assistant that compares resumes against job descriptions, calculates job-fit scores, identifies missing skills, and generates personalized recommendations, interview questions, and learning plans.

Overview

Resume-Meets-Job was built to help job seekers understand how well their resume matches a target job description. The app acts like an AI career coach by reviewing resume content, comparing it with job requirements, and producing structured feedback that users can act on.

Key Features

  • • Resume-to-job-description fit analysis
  • • Job-fit score generation
  • • Skill gap identification
  • • Tailored resume improvement recommendations
  • • Custom interview questions based on the job description
  • • Personalized learning plan for missing skills

Tech Stack

Python Streamlit OpenAI API Prompt Engineering LLM Workflows

What I Built

I designed the end-to-end application flow, built the Streamlit interface, created structured prompts for resume and job analysis, and connected the system to AI-generated outputs. The project focuses on making resume feedback practical, specific, and easy to understand.

Impact

This project shows my ability to build deployed AI tools that combine user input, LLM reasoning, structured output generation, and a clean user interface. It also demonstrates practical use of AI for career preparation and workflow automation.
