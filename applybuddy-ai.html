ApplyBuddy-AI

An AI-powered job application assistant that helps users generate tailored resumes, cover letters, recruiter emails, LinkedIn outreach messages, and interview preparation materials from a resume and job description.

Overview

ApplyBuddy-AI was built to reduce the time and effort required to prepare job application materials. Instead of manually rewriting resumes, cover letters, recruiter emails, and interview notes for every role, users can upload or paste their resume and job description, then generate personalized outputs using AI.

Key Features

  • • Tailored ATS-optimized resume generation
  • • Personalized cover letter creation
  • • Recruiter email and LinkedIn outreach message generation
  • • Job-specific resume optimization suggestions
  • • Interview talking points and preparation materials
  • • Downloadable AI-generated outputs

Tech Stack

Python Streamlit OpenAI API PyPDF python-docx Prompt Engineering

What I Built

I developed the application workflow, integrated OpenAI APIs, handled document-processing logic, and designed prompts that generate structured, job-specific outputs. The project focuses on practical automation: turning resume and job-description inputs into useful application assets that can be reviewed, edited, and submitted faster.

Impact

The project demonstrates my ability to build real-world AI tools using LLMs, APIs, document parsing, and user-facing application design. It also shows how AI can be applied to automate repetitive professional workflows while keeping the final output personalized and relevant.
